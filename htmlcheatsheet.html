HTML Cheat Sheet

HTML Tags

Tag Description
<!DOCTYPE html> Declares the document type and version of HTML
<html> Defines the root element of an HTML page
<head> Contains metadata and other information about the HTML page
<title> Defines the title of the HTML page, displayed in the browser's title bar
<body> Contains the visible content of the HTML page
<h1> to <h6> Defines headings of different levels, with <h1> being the most important and <h6> being the least important
<p> Defines a paragraph of text
<a> Defines a hyperlink to another web page or a specific location within the same web page
<img> Displays an image on the web page
<ul> Defines an unordered list of items
<ol> Defines an ordered list of items
<li> Defines a list item within an unordered or ordered list
<form> Defines a form for user input
<input> Defines an input field for user input, with various types such as text, email, password, checkbox, radio, etc.
<button> Defines a clickable button
<label> Defines a label for an input element, which improves accessibility for screen readers
<select> Defines a dropdown list of options for user selection
<option> Defines an option within a select element
<textarea> Defines a multi-line text input field
<div> Defines a container for other HTML elements, which can be styled with CSS

HTML Attributes

Attribute Description
id Specifies a unique identifier for an HTML element, which can be used to reference it in CSS or JavaScript
class Specifies one or more class names for an HTML element, which can be used to style it with CSS or select it with JavaScript
src Specifies the URL of the image or other media file to be displayed
alt Specifies alternative text to be displayed if the image or media file cannot be loaded or accessed
href Specifies the URL of the web page or location within the same web page to be linked to
target Specifies where the linked web page should be opened, such as in a new tab or window
style Specifies inline CSS styling for an HTML element
title Specifies additional information about an HTML element, which is displayed as a tooltip when the user hovers over it
disabled Disables an input element or button
readonly Makes an input element read-only, so that the user cannot edit its value
required Makes an input element required, so that the user must enter a value before submitting the form
