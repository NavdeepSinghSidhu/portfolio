CSS Cheatsheet

CSS Selectors

  • element selector - Selects all elements of a certain type
  • class selector - Selects elements with a specific class
  • id selector - Selects an element with a specific id
  • descendant selector - Selects elements that are descendants of a certain element
  • child selector - Selects elements that are direct children of a certain element
  • adjacent sibling selector - Selects the next sibling element
  • attribute selector - Selects elements with a specific attribute
  • pseudo-class - Selects elements based on their state or position
  • pseudo-element - Selects a specific part of an element

CSS Properties

  • color - Sets the color of text
  • font-size - Sets the size of the font
  • margin - Sets the margin around an element
  • padding - Sets the padding inside an element
  • background-color - Sets the background color of an element
  • display - Specifies how an element should be displayed
  • position - Sets the position of an element
  • width - Sets the width of an element
  • height - Sets the height of an element
  • border - Sets the border properties of an element
  • text-align - Sets the alignment of text within an element

CSS Units

  • px - Pixels
  • % - Percentage
  • em - Relative to the font-size of its closest parent or the element itself
  • rem - Relative to the font-size of the root element
  • vh - Relative to the height of the viewport
  • vw - Relative to the width of the viewport
  • pt - Points (1/72 of an inch)
  • cm - Centimeters
  • in - Inches
