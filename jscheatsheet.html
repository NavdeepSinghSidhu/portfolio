JavaScript Cheat Sheet

Basic Syntax

  • var - Declares a variable
  • let - Declares a block-scoped variable
  • const - Declares a constant variable
  • if - Executes a block of code if a condition is true
  • else - Executes a block of code if a condition is false
  • for - Loops through a block of code a specific number of times
  • while - Loops through a block of code while a condition is true
  • function - Declares a function
  • return - Returns a value from a function
  • typeof - Returns the data type of a variable

DOM Manipulation

  • document.getElementById() - Retrieves an element by its id
  • document.getElementsByClassName() - Retrieves elements by their class name
  • document.getElementsByTagName() - Retrieves elements by their tag name
  • document.querySelector() - Retrieves the first element that matches a CSS selector
  • document.querySelectorAll() - Retrieves all elements that match a CSS selector
  • element.innerHTML - Sets or retrieves the HTML content of an element
  • element.textContent - Sets or retrieves the text content of an element
  • element.setAttribute() - Sets the value of an attribute on an element
  • element.getAttribute() - Retrieves the value of an attribute from an element
  • element.classList.add() - Adds a class to an element
  • element.classList.remove() - Removes a class from an element
  • element.classList.toggle() - Toggles a class on/off on an element
  • element.addEventListener() - Adds an event listener to an element
  • event.preventDefault() - Prevents the default behavior of an event
  • event.target - Retrieves the element that triggered an event
  • event.type - Retrieves the type of an event
